# Tic-Tac-Toe на Go
Веб-приложение для игры в крестики-нолики против компьютера, реализованное на стандартной библиотеке `net/http`. Компьютер использует алгоритм **Minimax** для выбора оптимального хода.

## Особенности
- Игровая логика с проверкой корректности ходов и определением победителя
- Поддержка одновременных партий (идентификация по UUID)
- Потокобезопасное хранение состояния игр в памяти (`sync.Map`)
- Чистая архитектура: разделение на слои domain, datasource, web и DI
- Внедрение зависимостей через `uber/fx`

## API
`POST /game/{game_uuid}`
Принимает текущее состояние доски после хода игрока, возвращает доску с ответным ходом компьютера.
При некорректных данных возвращает ошибку с описанием.
